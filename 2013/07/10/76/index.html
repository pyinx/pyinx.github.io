安装glibc2.15的过程
2013-07-10 | pyinx | linux

 

系统版本： rhel 5.8 32位

内核版本：2.6.18-300.el5PAE

gcc版本：gcc version 4.4.6 20110731 (Red Hat 4.4.6-3) (GCC)

 

安装过程：

cd /usr/local/src

tar zxf binutils-2.22.tar.gz

cd binutils-2.22

./configure

make

make install

tar zxf glibc-2.15.tar.gz

tar zxf glibc-ports-2.15.tar.gz

mv glibc-ports-2.15 glibc-2.15/ports

mkdir glibc-build

cd glibc-build/

../glibc-2.15/configure –prefix=/usr/local/glibc CC=gcc44

make

make install
