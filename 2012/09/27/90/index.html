搜狗输入法 Linux 版本即将诞生
2012-09-27 | pyinx | linux

 

 

Linux 桌面用户一直渴望在 Linux 平台下有个好的输入法，看来这个愿望马上快要实现了。在 9 月 22 日举行的“Linux Deepin 2012 用户与开发者大会”上，搜狗桌面事业部高级经理杨磊为大家做了《输入法的互联网时代》的精彩演讲，“云计算”的运用使得输入法更新、更准。互联网成为输入法产品与用户的沟通载体。

杨磊表示：马上就会有与 Linux Deepin 合作的搜狗输入法 Linux 版本发布。

在本次大会中，Linux Deepin 开发负责人王勇也为大家展示了 搜狗输入法 Linux 版本的演示视频。

参与开源社区、推动 Linux 桌面的推广，并一定意味着对商业公司产品的强制开源，产品的决定权还是掌握在商业公司手中。希望商业公司在考虑推出 Linux 桌面产品的时候不要有所顾虑。搜狗输入法与 Linux Deepin 的友好合作为国内的商业公司在 Linux 平台产品的推出模式上起了一个很好的表率作用。
