php-fpm启动报错
2012-05-18 | pyinx | it故障


安装完php-fpm后 不能启动。

[root@localhost etc]# /usr/local/php/sbin/php-fpm start
Starting php_fpm .May 18 04:05:37.653467 [ERROR] fpm_unix_conf_wp(), line 124: please specify user and group other than root, pool ‘default’
……………………………. failed

出现些错误提示，请修改 php-fpm.conf

> 
                        Unix user of processes
>
>                 <!–    <value name=”user”>www</value>                          –>
>
>
>                         Unix group of processes
>
>                 <!–    <value name=”group”>www</value>         –>

换成（去掉注释）

> Unix user of processes
>
> <value name=”user”>www</value>
>
>
> Unix group of processes
>
> <value name=”group”>www</value>
修改完后，重新启动。

/usr/local/etc/rc.d/php-fpm start
Starting php_fpm. done
