ubuntu上为python安装matplotlib绘图工具时报错及解决方法
2012-01-05 | pyinx | python

在ubuntu 10.10上安装python会图工具-matplotlib时，报错：

error trying to exec ‘cc1plus’: execvp: 没有那个文件或目录

解决方法：

sudo apt-get install -y build-essential

然后继续执行python setup.py install 去安装matplotlib，又报错：

Python.h：没有该文件或目录

解决方法：

sudo aot-get install -y python2.7-dev (我安装的python是2.7版本的)

继续执行python setup.py install 去安装matplotlib，又报错：

ft2build.h：没有那个文件或目录

解决方法：

sudo apt-get install libxft2-dev

继续安装，继续报错：
src/_png.cpp:10:20: 致命错误： png.h：没有那个文件或目录

解决方法：
sudo apt-get install -y libpng12-dev

继续，报错：
权限不够
解决：
sudo python setup.py install

最后，终于安装成功了，真不容易啊。
