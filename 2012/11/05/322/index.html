python2.4升级到python2.7
2012-11-05 | pyinx | python

###

安装前确认zlib是否安装？

yum install zlib zlib-devel

wget http://www.python.org/ftp/python/2.7.3/Python-2.7.3.tgz

tar zxvf Python-2.7.3.tgz

cd Python-2.7.3

./configure –prefix=/usr/local/python2.7 && make && make install

cp /usr/local/python2.7/bin/python2.7 /usr/bin/

yum

安装完毕，但会发现yum需要系统自带的2.4版本，主要一定修改yum，否则会导致yum不能使用。

vi /usr/bin/yum

第1行

#!/usr/bin/python修改为：#!/usr/bin/python2.4

然后安装EasyInstall

wget http://pypi.python.org/packages/2.7/s/setuptools/setuptools-0.6c11-p2y2.7.egg

sh setuptools-0.6c11-p2y2.7.egg
安装完成后就可以直接使用easy_install命令安装第三方Python开发包了，例如：easy_install SQLObject
全部自动完成，不用一个个去找egg了。
