sed实用——文本转换
2011-08-22 | pyinx | linux

sed -e ‘s/$/\r/‘ file 将 UNIX 风格的文本转换成 DOS/Windows 格式。

您可能知道，基于 DOS/Windows 的文本文件在每一行末尾有一个 CR（回车）和 LF（换行），而 UNIX 文本只有一个换行。有时可能需要将某些 UNIX 文本移至 Windows 系统，该脚本将为您执行必需的格式转换。

在该脚本中，’$’ 规则表达式将与行的末尾匹配，而 ‘\r’ 告诉 sed 在其之前插入一个回车。在换行之前插入回车，立即，每一行就以 CR/LF 结束。请注意，仅当使用 GNU sed 3.02.80 或以后的版本时，才会用 CR 替换 ‘\r’。

通过sz将1（linux格式文本）和111（通过sed转换后的文本）传到windows上。

将把 DOS/Windows 格式的文本转换成可信赖的 UNIX 格式：

sed 's/.$//' dos.txt &gt;linux.txt
