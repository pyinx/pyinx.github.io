代码：

<!– BEGIN: GeoIPView.com IP Locator –>

<script type=”text/javascript” src=”
http://api.geoipview.com/api.php?t=1&amp;lang=cn&amp;w=230&amp;h=300&amp;bg=ECDDC0&amp;bd=8DADCC&amp;tx=222222”></script><br>

Powered by <a href=”http://cn.geoipview.com“ title=”IP 地理位置查询”>GeoIPView</a>

<!– END: GeoIPView.com IP Locator –>

代码：

<!– BEGIN: GeoIPView.com IP Locator –>

<script type=”text/javascript” src=”
http://api.geoipview.com/api.php?t=0&amp;lang=cn&amp;w=180&amp;h=52&amp;bg=ECDDC0&amp;bd=8DADCC&amp;tx=222222”></script><br>

<noscript><a href=”http://cn.geoipview.com“ title=”IP 地理位置查询”>http://cn.geoipview.com</a></noscript&gt;

<!– END: GeoIPView.com IP Locator –>
备注：你可以直接修改 JS 脚本 URL 中的下列定制参数:

  • w: 宽度
  • h: 高度
  • bg: RGB 值表示的背景颜色。例如：ECDDC0
  • bd: RGB 值表示的边框颜色。例如：8DADCC
  • tx: RGB 值表示的文字颜色。例如：0F0F0F
