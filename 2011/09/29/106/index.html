wdcdn的官方网站：http://www.wdlinux.cn/

Wdcdn是一套完整的CDN网页加速缓存系统,包括squid系统及web管理后台,可帮助中小站长或中小企业快速构建自己的CDN网络及服务器群,提供更好的服务,更快速的网站,我们也致力打造这样一个完善完美的CDN系统
分单节点版本,多节点(集中管理版本)两个版本
1 单节点版本就是用一台或两台机,单独配置squid和wdcdn管理后台,此方式适用于中小站点或企业,构建自己的CDN网络,最简单的就比如一个电信,一个网通,然后加上DNS解释就搞掂
2 多节点,也就是集中管理版本,是针对专门提供CDN服务的公司和企业,具有N多台服务器,带宽等资源,集中一个管理后台,可管理所有服务器上的配置,统计等集中式管理和配置,方便易操作!

由于多节点wdcdn是需要money的，所以只能搭建单节点的wdcdn进行测试了；

首先需要安装wdcp管理系统，安装非常简单，在Centos目录下wget下wdcp的安装脚本rpm_install,然后执行该脚本即可：

wget wget http://dl.wdlinux.cn:5180/rpms/rpm_install
sh  rpm_install
其中会让你选择服务器的架构是apache还是nginx：
Select Install
1 apache + php + mysql + zend + eAccelerator + vs[ftp](http://www.linuxso.com/command/ftp.html)d + phpmyadmin + wdcp
2 nginx + php + mysql + zend + eAccelerator + vsftpd + phpmyadmin + wdcp
3 don't install is now
Please Input 1,2,3: 2
安装完成后，可以输入IP地址进行访问：
onfiguration ,lamp or [ln](http://www.linuxso.com/command/ln.html)mp,wdcp install is finsh[ed](http://www.linuxso.com/command/ed.html)
 visit http://ip
 wdcp visit http://ip:8080
 [more](http://www.linuxso.com/command/more.html) infomation please visit http://www.wdlinux.cn
安装完wdcp管理系统后，就可以安装wdcdn了，步骤和安装wdcp类似：
wget http://www.wdlinux.cn/in_scripts/wdcdn_install
sh  wdcdn_install
安装完成wdcdn后，在wdcp的后台管理可以看到：
![](http://www.linuxso.com/uploads/allimg/110928/0P40MU9-0.jpg "wdcdn1") 
在“网站管理”栏目里添加需要加速的域名和所对应的IP地址即可：
指本地hosts文件进行访问，可以看到wdcdn的缓存情况。
wdcdn的缓存信息可以在“状态统计“中看到。
